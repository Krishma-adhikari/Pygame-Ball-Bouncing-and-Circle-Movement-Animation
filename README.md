# Pygame-Ball-Bouncing-and-Circle-Movement-Animation
## Description

This project contains basic animations created using Python and Pygame.
It includes simple examples like:
1. Ball bouncing animation
2. Circle movement animation

These are created to understand basic motion and game loop concepts in Pygame.
## Requirements
1. Python
2. Pygame
## Features
1. Simple ball bouncing inside screen
2. Moving circle animation
3. Basic use of Pygame game loop
4. Collision with screen edges

## Purpose
This project is made for learning:
1. Basic animation in Pygame
2. Movement using coordinates
3. Game loop structure
