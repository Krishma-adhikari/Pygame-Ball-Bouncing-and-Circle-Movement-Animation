#  🎮 Learning Animation + Coding with Pygame

## 🚀 Description
Excited to start my journey in animation + coding! I’m discovering how Python and Pygame can bring imagination to life through motion and visuals ✨

This project focuses on understanding basic movement, animation, and game loop concepts in Pygame.

---

## ⚙️ Requirements
- Python  
- Pygame  

---

## ✨ Features
- 🟠 Simple ball bouncing inside the screen  
- 🔵 Moving circle animation  
- 🔁 Basic Pygame game loop implementation  
- 💥 Collision with screen edges  
- 🚗 Simple car movement from left to right  
- 🟠Collecting Balls by moving rectangle.
---

## 🎯 Purpose
This project was built for learning:
- Basic animation in Pygame  
- Movement using coordinates  
- Game loop structure  
- Understanding real-time rendering
